# DegraTwin

DegraTwin is a Streamlit-based digital twin tool for textile degradation simulation.

The application uses humidity and temperature data to evaluate environmental risk for selected textile materials and simulate degradation under different environmental scenarios.

## Main features

- Load sensor readings from an external API
- Evaluate environmental risk for selected textile materials
- Simulate degradation under different scenarios
- Display plots and preview tables
- Export a JSON summary of the simulation
